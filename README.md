## Naive paper downloader from official site

### Usage 
```javascript
node {target}.js {output directory} {url}
```
Supported targets: JMLR, KDD

ex:
ICML 2016
``` javascript
node JMLR.js ./download http://jmlr.org/proceedings/papers/v48/
```
